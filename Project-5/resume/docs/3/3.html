How to configure IP address
Open cmd prompt-type(ncpa.cpl)-LAN-Right
Click-Properties-Choose(IPv4)-Properties.
DEFINITION:
● It is a value used to identify the network bits and host bits in
the given IP address by the computer.
● It is represent in “Dotted Decimal Notation”.
● It is mandatory parameter.
FORMULA:
Apply once in network bits and apply zeros in host bits to find subnet
mask
