Linux commands to know
1. Sudo -i->to login as root user.
2. apt-get update->to update the latest version.
3. clear->to clear the page
4. mkdir azure->to create folder named azure
5. ls-> to list the folders or files
6. cd azure->go inside the folder or file
7. touch file1.txt->to create file named file1.
8. vi file1.txt->to modify the txt file->after press i and type anything to edit and press ESC.
9. :wq->to save the file or folder.
10. cat file1.txt->to view the edited file.
11. cd ..->to exit.
12. apt-get install apache2 -y->to install page.
13. cd /var/www/html->to open the apache file.
14. vi index.html->to edit the html page.-> scroll till 198 page->edit apache as anything.
15. Press ESC and type :wq.
